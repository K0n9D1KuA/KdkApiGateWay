# KdkApiGateWay
基于 Netty，Nacos，NacosConfig 开发的一款开箱即用的“微内核+插件”架构的 API 网关，可配置接口粒度的网关路由规则，可实时热更新路由规则。具有负载均衡， 多维度限流，MOCK 模拟，灰度发布，多维度黑名单限制，短链系统，超时重试，服务熔断降级等功能。
